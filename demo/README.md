# Demo Assets

Add screenshots and demo media here:

- `screenshot.png` — Main app screenshot (referenced in README.md)
- `demo.gif` or `demo.mp4` — Optional walkthrough recording

## How to capture

1. Run the app: `streamlit run app.py`
2. Select a sample JD and click Analyze
3. Take a screenshot of the full results page
4. Save as `screenshot.png` in this folder
